# Universal Converter

Корневой сайт проекта переведен в первый рабочий срез Universal Converter: теперь это единый статический SPA-shell, в котором категории подключаются через конфигурацию и адаптеры, а не через жестко зашитый интерфейс под один тип величины.

## Что уже работает

- Категория Давление на общем движке конвертации через базовую единицу Па.
- Категория Температура на том же UI через отдельный формульный адаптер.
- Верхняя навигация между категориями без перезагрузки страницы.
- Переключение темы.
- Универсальные карточки результатов с копированием.
- Заглушка для будущего валютного модуля.

## Текущий стек

- HTML5
- JavaScript ES modules
- Tailwind CSS через CDN
- Статический запуск без сборки и без собственного бэкенда

## Архитектурный принцип

Каждая категория описывается через:

1. конфиг с метаданными и списком единиц;
2. адаптер преобразования;
3. общий UI, который рендерит ввод, выбор исходной единицы и карточки результатов.

Этот подход нужен для последующего подключения настроек, localStorage-состояния, валютных API и новых физических категорий без переписывания экрана.

## Локальный запуск

Откройте корневой файл index.html в браузере.

## Ближайшие шаги

1. Подключить модальное окно настроек и сохранение активных карточек.
2. Сохранить тему, категорию и исходную единицу в localStorage.
3. Интегрировать валютный модуль на основе существующего course-convertor.
